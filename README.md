# stop-slop-ru

Скилл для Claude Code, который чистит русский текст от следов нейросети.

Это русская версия [stop-slop](https://github.com/hardikpandya/stop-slop) (Hardik Pandya). Английские правила, которые в русском не работают, я убрал. Вместо них — маркеры, которых в оригинале нет: канцелярит, копула, синонимическая карусель, деепричастные цепочки. Плюс рабочий линтер, которого в оригинале тоже нет.

> [!IMPORTANT]
> Это гонка вооружений, и маркеры устаревают. Как только признак ИИ-текста описывают публично, модели учатся его обходить — в исходных статьях это прямо называют бесконечной гонкой. Считай, что через год половину корпуса придётся переписать. Скилл живёт, только пока его обновляют: новый штамм заметил — завёл маркер в `markers.yaml`, поднял версию. Без этого он медленно протухнет.

## Установка

Скопируй папку `stop-slop-ru` в один из каталогов:

- `~/.claude/skills/` — скилл работает во всех проектах;
- `.claude/skills/` — только в текущем проекте.

Claude Code подхватит его сам.

## Как пользоваться

Скилл включается сам, когда пишешь, правишь или проверяешь русский текст. Можно позвать и вручную: «почисти этот текст от слопа» или «проверь на ИИ-маркеры».

Есть и отдельный линтер — проверить файл без Claude. Запускай из папки скилла:

```sh
python3 scripts/stopslop.py text.md                       # проверить весь текст
python3 scripts/stopslop.py text.md --severity high       # только явные маркеры
python3 scripts/stopslop.py text.md --category phrases     # одна категория
python3 scripts/stopslop.py text.md --id em-dash-overuse   # один маркер
python3 scripts/stopslop.py --list                         # список всех маркеров
```

Нужен только Python 3, без установки пакетов. Для CI: `--min-score N` вернёт код 1, если маркеров больше N.

## Что внутри

- `SKILL.md` — правила, быстрые проверки, шкала оценки.
- `markers.yaml` — все маркеры в одном файле (сейчас 28). Отсюда их берёт и человек, и линтер.
- `scripts/stopslop.py` — линтер.
- `references/` — словарь фраз, разбор структур, примеры «было / стало».
- `CONTRIBUTING.md` — как добавить новый маркер.
