# phrases.md. Справочник слов и фраз к удалению

Словарь маркеров AI-слопа для русского языка. Нашёл фразу из списка — почти всегда её надо вырезать или заменить на прямую формулировку. Замены даны как ориентир, не как механическая подстановка: иногда правильный ход — удалить предложение целиком.

Здесь только лексика и фразовые штампы. Синтаксис (плоская вариативность длины, гиперсвязность, пассив, водянистость), а также скоринг по осям — в `references/structures.md` и в `SKILL.md`. Деепричастные цепочки и синонимическая карусель попадают в оба файла: фразовые признаки даны ниже как мягкие флаги, разбор по синтаксису — в `structures.md`.

Колонка «контекст» в §2, §8 и §14 помечает полисемичные слова: запрет действует только в указанном значении, не на слово вообще.

## 1. Пустые открытия (вступления-пустышки)

Фразы, которыми текст «разогревается» вместо того, чтобы начать с дела. Первое предложение должно нести факт, а не настроение.

| Избегать | Заменить на |
|---|---|
| В современном динамично развивающемся мире… | (вырезать; начать с конкретного факта) |
| В современном мире / В наше время / В нынешних реалиях | (вырезать) |
| На сегодняшний день | (вырезать; нужна дата — назвать её) |
| В эпоху цифровизации / В эпоху цифровых технологий / В цифровую эпоху | (вырезать) |
| В условиях постоянно меняющегося рынка | (вырезать; назвать конкретное изменение) |
| Сложно представить современный мир без… | (вырезать) |
| Трудно переоценить (роль / значение) | (вырезать; сразу сказать, в чём роль) |
| Издавна / С давних времён / Испокон веков | (вырезать или назвать период) |
| Все мы знаем, что… / Каждый из нас сталкивался с… | (вырезать) |
| Не секрет, что… / Ни для кого не секрет | (вырезать) |
| В последнее время всё чаще… | (назвать срок и источник) |
| Сегодня как никогда / Сейчас, как никогда раньше | (вырезать) |
| Мир не стоит на месте | (вырезать) |
| Начнём с того, что… / Прежде всего стоит сказать | (вырезать; сказать сразу) |

## 2. Канцелярит и отглагольные существительные

Главный токсичный паттерн русского AI-текста. Существительное на **-ание / -ение / -ация / -ность / -ствление** почти всегда прячет глагол. Найди действие, верни ему глагол.

| Канцелярит | Живой глагол | Контекст |
|---|---|---|
| осуществить внедрение | внедрить | |
| осуществлять контроль / производить контроль | контролировать, проверять | |
| в целях оптимизации | чтобы ускорить / упростить | |
| с целью повышения эффективности | чтобы работать лучше | |
| произвести оплату | оплатить, заплатить | |
| осуществить отправку | отправить | |
| принять решение о | решить | |
| оказывать влияние на | влиять на | |
| вести деятельность по | заниматься, делать | |
| в рамках реализации проекта | в проекте / когда делаем проект | |
| осуществление деятельности | работа | |
| обеспечение выполнения | добиться выполнения / проследить, чтобы выполнили | не равно «выполнить»: речь о чужом выполнении |
| проведение анализа | проанализировать, разобрать | |
| дать оценку | оценить | |
| оказать поддержку | поддержать, помочь | |
| осуществить разработку | разработать | |
| в случае возникновения необходимости | если понадобится | |
| при наличии возможности | если получится / по возможности | |
| в связи с отсутствием | потому что нет | |
| на постоянной основе | постоянно | |
| в большом количестве | много | |
| имеет своей целью | нужен для того, чтобы | |
| является следствием | следует из / поэтому | |
| путём использования | с помощью / используя | |

Эмпирическое правило: два-три слова на **-ение/-ание/-ация** подряд — переписывай.

## 3. Копулятивная аллергия (бегство от слова «это»)

Модель избегает простой связки и подменяет её тяжёлыми оборотами. В 90% случаев чинится словом «это» или прямым глаголом.

| Избегать | Заменить на |
|---|---|
| представляет собой | это |
| является (в роли связки) | это / прямой глагол |
| служит основой / служит инструментом | основа / инструмент (без «служит») |
| выступает в качестве | это / работает как |
| имеет место быть | есть / бывает / происходит |
| носит характер (системный, регулярный) | системный, регулярный (прилагательным) |
| представляет интерес | интересно / важно |
| находит своё отражение в | видно в / отражается в |
| обладает способностью | может / умеет |
| характеризуется (наличием) | у него есть / для него характерно |

Проверка: замени оборот на «это». Смысл не пострадал — оборот был лишним. Тире-связку (Москва — столица) тоже можно ставить, но это обычная грамматика, не драматическая пауза; про переизбыток тире — §13.

## 4. Овер-хеджирование (лишние оговорки и важничанье)

Дежурные фразы, которыми текст набивает себе вес. Информации не несут: читатель и так понимает, что дальше важное. Почти всегда вырезаются целиком.

| Избегать |
|---|
| Важно отметить, что… |
| Стоит отметить / Следует отметить |
| Стоит подчеркнуть / Необходимо подчеркнуть |
| Следует учитывать / Нужно учитывать |
| Нельзя не заметить / Нельзя не отметить |
| Необходимо понимать / Важно понимать |
| Крайне важно / Особенно важно |
| Хочется отметить / Хотелось бы отметить |
| Как известно / Как всем известно |
| Стоит сказать, что… |
| Заметим, что… / Отметим, что… |
| Надо сказать, что… |
| Что немаловажно |
| Не лишним будет упомянуть |

Правило: удали оговорку, оставь само утверждение. «Важно отметить, что цена выросла» → «Цена выросла».

## 5. Модальное хеджирование (лишняя модальность)

Текст прячет утверждение за «может / способен / позволяет». Действие реально происходит — пиши прямо. Модальность оставляй только там, где речь о настоящей возможности или риске.

| Избегать | Заменить на |
|---|---|
| может стать (хорошим решением) | станет / (прямо назвать) |
| способен обеспечить | обеспечивает / даёт |
| позволяет (в каждом предложении) | прямой глагол: «позволяет ускорить» → «ускоряет» |
| призван решить / призван обеспечить | решает / нужен для |
| даёт возможность | прямой глагол: «даёт возможность настроить» → «настраивает» |
| способствует (повышению, улучшению) | повышает, улучшает |
| имеет потенциал | может (если правда может) / вырезать |
| в перспективе может | (назвать срок и условие или вырезать) |

Проверка: «X позволяет ускорить процесс» = «X ускоряет процесс». Правда ускоряет — убери «позволяет».

## 6. Промо-регистр и пустые эпитеты

Рекламные прилагательные без цифр и фактов. Либо подставь конкретику, либо вырежи: пустой эпитет всегда слабее факта.

| Избегать | Заменить на |
|---|---|
| уникальный | (чем именно отличается — конкретно) |
| инновационный | (что нового — назвать) |
| ведущий / один из ведущих | (доля рынка, место в рейтинге — цифра) |
| потрясающий / впечатляющий / удивительный | (вырезать; показать факт) |
| не имеющий аналогов / не имеет аналогов | (вырезать или доказать) |
| высококачественный / высокотехнологичный | (назвать стандарт, материал, технологию) |
| широкий спектр / широкий ассортимент | (сколько именно / какие) |
| передовой / самый современный | (вырезать или назвать версию/год) |
| незаменимый помощник | (вырезать) |
| расположенный в самом сердце (города) | (адрес / район) |
| идеальный выбор / идеальное решение | (для какой задачи — конкретно) |
| полностью / абсолютно / максимально (+эпитет) | (вырезать усилитель) |
| настоящий / поистине / по-настоящему | (вырезать) |
| богатый опыт / огромный опыт | (сколько лет / проектов) |
| на высшем уровне | (по какому критерию) |

## 7. Англо-кальки (буквальный перенос английских клише)

Английские штампы, перенесённые в русский дословно. Звучат «переведённо». В скобках — английский оригинал, чтобы узнавать в лицо.

| Избегать | Заменить на |
|---|---|
| играет ключевую / важную / решающую роль *(plays a pivotal role)* | важен / определяет / нужен для |
| в конце концов / в конечном счёте *(at the end of the day)* | (вырезать) |
| в заключение / подводя итог *(in conclusion)* | (вырезать; закончить мыслью) |
| в связи с этим *(in this regard)* | поэтому |
| тем не менее *(however, в начале каждого абзаца)* | но / однако (редко, не в каждом абзаце) |
| более того / кроме того *(moreover, в каждом абзаце)* | (прореживать; убрать ковровые связки) |
| это меняет правила игры *(game changer)* | (вырезать; сказать, что именно меняется) |
| на этой ноте *(on that note)* | (вырезать) |
| давайте погрузимся / давайте углубимся *(let's dive in)* | (вырезать; начать тему) |
| когда дело доходит до *(when it comes to)* | в / для / насчёт |
| не только… но и *(not only… but also, как штамп)* | (часто проще перечислить через «и») |
| в мире (чего-то) *(in the world of)* | (вырезать: «в мире финансов» → «в финансах») |
| вооружившись (знаниями / инструментами) *(armed with)* | (вырезать или прямо: «зная…») |
| секрет (успеха) кроется в *(the secret lies in)* | (вырезать; назвать причину) |
| вывести (бизнес) на новый уровень *(take it to the next level)* | (вырезать; назвать конкретный результат) |

## 8. Семантические кальки (English-bias: слова формально верные, но смещённые)

Самый коварный класс. Слово существует в русском, но выбрано под влиянием английского значения и звучит не по-русски. Часто это калька через английский корень или сочетаемость.

| Избегать (смещено) | По-русски | Контекст |
|---|---|---|
| основание науки / основание теории | основы науки / основы теории | |
| маркетинговые усилия *(marketing efforts)* | маркетинг / работа над маркетингом | |
| уточните (доработайте) усилия *(refine your efforts)* | доработайте подход / доведите до ума | |
| адресовать проблему *(address the issue)* | решить проблему / заняться проблемой | |
| контролировать *(to control = руководить)* | управлять / распоряжаться | только в значении «управлять»; «контролировать качество» — норма |
| реализовать *(realize = осознать)* | понять / осознать | только в значении «осознать»; «реализовать проект» — норма |
| опыт *(experience = впечатление)* | впечатление / ощущение | «пользовательский опыт» — норма; «получить опыт от похода» — нет |
| доставить ценность *(deliver value)* | принести пользу / быть полезным | |
| на ежедневной основе *(on a daily basis)* | ежедневно / каждый день | |
| сильное мнение *(strong opinion)* | твёрдое мнение / уверен, что | |
| это делает смысл *(it makes sense)* | это логично / в этом есть смысл | |
| взять решение *(take a decision)* | принять решение | |
| провести время качественно *(quality time)* | побыть вместе / провести время с пользой | |

## 9. Дидактический / учительский тон

Текст ведёт читателя за руку и комментирует собственную структуру. Переходи сразу к делу: читатель и сам видит, что ты «рассматриваешь» тему.

| Избегать | Заменить на |
|---|---|
| Давайте рассмотрим / Давайте разберём | (сразу разбирать) |
| Разберём подробнее / Рассмотрим подробнее | (разбирать) |
| Как мы видим / Как видно из… | (вырезать; вывод и так виден) |
| Стоит понимать, что… / Нужно понимать | (вырезать) |
| Обратите внимание, что… *(если не реальное предупреждение)* | (вырезать) |
| Как уже было сказано / Как отмечалось выше | (вырезать или дать ссылку коротко) |
| Прежде чем перейти к…, важно… | (перейти) |
| Поговорим о / Перейдём к | (перейти молча) |
| Что это значит? Давайте разбираться | (сразу объяснять) |

## 10. Псевдо-терапевтический / коучинговый регистр

Имитация эмпатии и «поддержки». Уместно только в реально личном, доверительном контексте (личный блог, письмо). В справке, статье, описании продукта — вырезать.

| Избегать |
|---|
| Это нормально, что ты так чувствуешь / Это абсолютно нормально |
| Ты не ошибаешься, что так чувствуешь |
| Услышь себя / Прислушайся к себе |
| Тихое подтверждение / Мягкое напоминание |
| Дай себе разрешение… / Позволь себе… |
| Ты не одинок(а) в этом |
| Будь добр(ее) к себе / Не вини себя |
| Твои чувства важны / Твои чувства валидны |
| Найди свой ритм / Найди то, что откликается |
| Сделай глубокий вдох |
| Помни: ты делаешь всё, что можешь |
| Это твоё путешествие / Это твой путь |
| Просто будь собой |

Контекст не личный — удалить блок целиком; редко — заменить одной конкретной рекомендацией.

## 11. Мета-комментарии (текст о тексте)

Текст рассказывает, что он сейчас делает, вместо того чтобы это делать. Почти всё вырезается без потерь.

| Избегать | Заменить на |
|---|---|
| В этой статье мы (рассмотрим / расскажем) | (сразу к сути) |
| Ниже разберём / Ниже представлены | (разбирать / показывать) |
| В данной статье / В данном материале / В рамках этой статьи | (вырезать) |
| Как мы увидим далее / Как будет показано ниже | (вырезать) |
| Об этом — далее / Подробнее об этом ниже | (вырезать или дать сразу) |
| Подводя итог / Подытоживая / Итак, подведём итоги | (вырезать; закончить выводом по делу) |
| В заключение хочется сказать | (вырезать) |
| Надеемся, эта статья была полезной | (вырезать) |
| Стоит начать с того, что | (начать) |
| Эта тема заслуживает отдельного внимания | (вырезать или раскрыть) |

Заголовки-пустышки к удалению/переименованию: **«Заключение», «Итоги», «Подводя итог», «Вместо заключения», «Несколько слов о…», «Введение»** — заменить на содержательный заголовок или убрать.

## 12. Конструкция «не X, а Y» (псевдо-противопоставление)

Русский аналог английского «not X, it's Y». По брифу встречается в 80%+ AI-текстов, часто по разу на абзац. Модель ставит ложный контраст, чтобы фраза звучала «глубоко», хотя X никто и не утверждал. Вырезай или переписывай прямым тезисом.

| Избегать | Заменить на |
|---|---|
| Это не просто инструмент, это… | (сказать прямо, что это) |
| Дело не в скорости, а в… | (назвать, в чём дело, без отрицания) |
| Не баг, а фича | (вырезать как штамп; описать поведение) |
| Это не про деньги, это про… | (сказать, про что, прямым утверждением) |
| Не цель, а средство / Не результат, а процесс | (сказать, чем является) |
| Важно не что, а как / Важно не сколько, а как | (сформулировать критерий прямо) |
| Не «X», а «Y» (через кавычки, как откровение) | (одно прямое определение) |

Правило: оставь только Y, поданный как обычное утверждение. «Это не просто тренажёр, это образ жизни» → «Тренажёр входит в привычку». Псевдо-противопоставление само по себе — маркер, даже если Y верный.

## 13. Имитация вовлечённости (псевдо-динамика)

Текст подделывает живой ритм рублеными цепочками и риторическими вопросами без ответа по делу. Создаёт ощущение динамики на пустом месте.

| Избегать |
|---|
| Коротко. Точно. По делу. (цепочки односоставных «для ритма») |
| Просто. Быстро. Удобно. |
| Зачем? Потому что… (псевдо-сократический заход) |
| Что дальше? Разбираемся. / А теперь — самое интересное. |
| Думаешь, это всё? Нет. |
| Звучит сложно? На деле нет. |
| И знаешь что? |

Правило: убери односоставную цепочку и риторический вопрос, оставь само содержание одной нормальной фразой. Один-два таких приёма на длинный текст — живой голос; цепочкой подряд — машинная имитация.

## 14. Искусственное словообразование (флаг для проверки)

Не строгий запрет, а сигнал перечитать. AI любит склеивать новые прилагательные и термины, которых нет в живой речи. Слово звучит как «сделанное на ходу» — проверь, есть ли оно в норме, замени на обычное.

| Подозрительное | Обычно лучше | Контекст |
|---|---|---|
| нейросетевой (как украшение) | «нейросетевой подход» → «подход с нейросетью» или вырезать | термин по делу — норма |
| маркетингово-аналитический | маркетинговый и аналитический (раздельно) | |
| дизайнеровский | дизайнерский | |
| клиентоориентированность (нагромождение) | «мы внимательны к клиентам» | |

## 15. Мягкие флаги: деепричастные цепочки и синонимическая карусель

Фразово-распознаваемые паттерны. Не вырезай по слову, а перечитай конструкцию целиком. Подробный синтаксический разбор — `references/structures.md`.

**Деепричастные нагромождения.** Два и больше деепричастных оборота, навешанных на один глагол. У AI встречаются в 2–5 раз чаще, чем у людей.

- Флаг: «развивает направления, обеспечивая рост, привлекая инвесторов и укрепляя позиции».
- Чинится так: разбей на отдельные предложения с глаголами. «Компания развивает направления. Растёт, привлекает инвесторов, укрепляет позиции».

**Синонимическая карусель.** Один и тот же объект в соседних предложениях называют разными словами, чтобы избежать повтора (срабатывает repetition penalty модели). Человек спокойно повторит слово или поставит местоимение.

- Флаг: компания → организация → предприятие → бренд; редактор → специалист → профессионал → автор.
- Чинится так: выбери одно слово и повторяй его (или местоимение). Повтор честнее карусели синонимов.

## Как пользоваться

1. Прогони текст по разделам 1–14: каждое совпадение — кандидат на рез.
2. Не подставляй замену механически — часто лучший вариант пустая строка целиком.
3. Раздел 14 — мягкий флаг словообразования, раздел 15 — мягкие флаги синтаксиса: не вырезай слепо, перечитай и упрости.
4. Прореди оформление: декоративные эмодзи и эмодзи-буллеты (⚡🎯💡✅🚀) — убрать; болд в каждом предложении — оставить один-два на абзац; тройные перечисления и тройную анафору («без шума, без хаоса, без перегруза») — разбить или расширить до иного числа.
5. Проверь плотность тире. Драматическое тире для паузы и нагнетания — убрать; тире-связка и тире на месте пропущенного слова — оставить. У машины тире втрое больше нормы (≈10 против ≈3 на 1000 слов).
6. Проверь ритм: не идут ли подряд три коротких рубленых предложения. Чередуй длину.
